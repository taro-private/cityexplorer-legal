# CityExplorer Privacy Policy / プライバシーポリシー

**Last updated / 最終更新: 2026-07-21**

---

## English

### Data We Do NOT Collect

CityExplorer does not collect, store, or transmit any personal information, account information, or payment information from players.

### Data Stored Locally on Your PC

| Data | Storage Location | Transmitted Externally? |
|------|-----------------|------------------------|
| Game settings, favorites, custom markers, scores | `%LOCALAPPDATA%\CityExplorer\userdata.dat` (AES-256 encrypted) | No |
| Developer-managed API keys | `CityExplorer\API_Key\apikeys.dat` (AES-256 encrypted, included in package) | No |
| Error logs | `Saved/ErrorLogs/` | No |
| Auto screenshots | `Saved/Screenshots/` | No |

All game data is stored exclusively on your local PC and is never sent to external servers.

### External Service Connections

During gameplay, the app connects to the following external services to provide map, weather, and facility information. Data transmitted is limited to **geographic coordinates (latitude/longitude)** and **search keywords**.

| Service | Purpose | Privacy Policy |
|---------|---------|----------------|
| Google Maps Platform (3D Tiles) | 3D city model rendering | [policies.google.com](https://policies.google.com/privacy) |
| OpenStreetMap / Overpass API | Map data, facility search | [osmfoundation.org](https://osmfoundation.org/wiki/Privacy_Policy) |
| OpenWeatherMap | Real-time weather data | [openweathermap.org](https://openweathermap.org/privacy-policy) |
| YOLP (Yahoo! Japan) | Domestic (Japan) facility search | [yahoo.co.jp](https://privacy.yahoo.co.jp/) |
| Google Gemini (Vertex AI) | AI-generated regional information | [policies.google.com](https://policies.google.com/privacy) |
| Wikipedia / Wikimedia | Regional encyclopedic information | [wikimediafoundation.org](https://foundation.wikimedia.org/wiki/Privacy_policy) |
| Open-Elevation | Elevation data | [open-elevation.com](https://api.open-elevation.com/) |

API keys used for service authentication are managed by the Developer and are not collected from or shared with users.

### AI-Generated Content

Regional information shown on the loading screen may be generated by Google Gemini. The Developer does not warrant the accuracy of AI-generated content.

### Contact

For questions regarding this Privacy Policy, please contact:  
📧 yokata.forest.words@gmail.com

## 日本語

### 収集しないデータ

CityExplorer は、プレイヤーの個人情報・アカウント情報・決済情報を一切収集・保管・送信しません。

### お使いの PC にローカル保存されるデータ

| データ | 保存場所 | 外部送信 |
|--------|----------|----------|
| ゲーム設定・お気に入り・自作マーカー・スコア | `%LOCALAPPDATA%\CityExplorer\userdata.dat`（AES-256暗号化） | なし |
| 開発者管理 API キー | `CityExplorer\API_Key\apikeys.dat`（AES-256暗号化・パッケージ同梱） | なし |
| エラーログ | `Saved/ErrorLogs/` | なし |
| 自動スクリーンショット | `Saved/Screenshots/` | なし |

全てのゲームデータはお使いの PC にのみ保存され、外部サーバーには送信されません。

### 外部サービスへの通信

ゲームプレイ中、地図・天候・施設情報の提供のために以下の外部サービスに接続します。  
送信されるデータは**現在地の緯度・経度**と**検索キーワード**に限定されます。

| サービス | 用途 | プライバシーポリシー |
|---------|------|------------------|
| Google Maps Platform（3D Tiles） | 3D都市モデル描画 | [policies.google.com](https://policies.google.com/privacy) |
| OpenStreetMap / Overpass API | 地図データ・施設検索 | [osmfoundation.org](https://osmfoundation.org/wiki/Privacy_Policy) |
| OpenWeatherMap | リアルタイム天候データ | [openweathermap.org](https://openweathermap.org/privacy-policy) |
| YOLP（Yahoo! Japan） | 国内施設検索 | [yahoo.co.jp](https://privacy.yahoo.co.jp/) |
| Google Gemini（Vertex AI） | AI地域情報生成 | [policies.google.com](https://policies.google.com/privacy) |
| Wikipedia / Wikimedia | 地域の百科事典情報 | [wikimediafoundation.org](https://foundation.wikimedia.org/wiki/Privacy_policy) |
| Open-Elevation | 標高データ | [open-elevation.com](https://api.open-elevation.com/) |

サービス認証に使用する API キーは開発者が管理し、ユーザーから取得・共有することはありません。

### AI生成コンテンツ

ローディング画面で表示される地域情報は、Google Gemini（Vertex AI）により生成される場合があります。AI生成コンテンツの正確性について、開発者は保証しません。
